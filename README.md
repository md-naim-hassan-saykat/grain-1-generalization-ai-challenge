# Grain 1 – Generalization AI Challenge

This repository contains the **starting kit and materials** for **Group 1 (Grain – Generalization)** of the AI-Master Challenge Course (2025–26)** at Université Paris-Saclay.

---

## Challenge Overview
The goal of this challenge is **grain variety classification**, with a strong focus on **model generalization**:
how well a model trained on one distribution performs on unseen or shifted data.

## Getting Started
1. Navigate to the [`Starting_Kit`](Starting_Kit/) folder  
2. Open `grain_1_generalization_starting_kit.ipynb`
3. Run all cells to explore the baseline pipeline

The notebook is designed to be **self-contained** and easy to extend.

---

## Evaluation Platform
This challenge will be deployed and evaluated using **Codabench**.  
The starting kit notebook is designed to be compatible with Codabench submissions.
